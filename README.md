# Ask Me Anithing - React + TypeScript + Vite

## Instalação do Vite
```
npm install vite@latest
```

## Instalação das dependências
```
npm i
```

## Executar a aplicação
```
npm run dev
```

## Instalação do TailwindCSS e suas bibliotecas
```
npm i tailwindcss postcss autoprefixer -D
```

## Inicializar o TailwindCSS
```
npx tailwindcss init -p
```

## Instalação da biblioteca React Router Dom
```
npm i react-router-dom
```

## Obter ícone do botão
```
npm i lucide-react
```

## Instalação do React 19 e do React Router Dom
```
npm install --save-exact react@rc react-dom@rc
```

## Forçar a instalação das dependências
```
npm i -f
```

## Instalação do Sonner
```
npm install sonner
```

## Forçar a instalação do Sonner
```
npm install sonner -f
```

## Instalação do Tanstack
```
npm i @tanstack/react-query -f
```
